# Knowledge

Base documental detalhada da Blue Ocean para consulta pelo agente.

Aqui mora a referência recorrente. O canon principal continua em `company/` e `governance/`.

## Áreas atuais
- `company-brain/` — leitura condensada do brain e regras de fonte de verdade
- `benchmarks/` — benchmarks internos úteis para leitura comparativa
- `patterns/` — padrões diagnósticos nomeados
- `matrices/` — ownership, governança e gatilhos de escalonamento
- `memory-governance.md` — política para separar memória local de inteligência versionada

## Regras
- conhecimento detalhado mora aqui
- playbooks e subagents devem apontar para esta base, não duplicá-la
- arquivos canônicos de negócio e governança vivem fora daqui, com referências cruzadas
- knowledge ajuda a interpretar, mas não substitui a fonte mais próxima do evento real
