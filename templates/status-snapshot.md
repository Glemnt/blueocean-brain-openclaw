# Template — Status Operacional Rápido

Use para abrir o dia, retomar uma frente ou dar visão executiva enxuta.

## Formato

```markdown
## Status rápido — [conta/frente]

**Período:** [recorte]
**Owner:** [pessoa/área se houver]
**Status:** saudável / atenção / risco / bloqueado
**Confiança:** [Alta/Parcial/Frágil/Insuficiente]

### 1. O que mudou
- [mudança 1]
- [mudança 2]

### 2. Sinais bons
- [sinal]

### 3. Riscos
- [risco + owner]

### 4. Decisões necessárias
- [decisão]

### 5. Próximas 24–48h
- [ação + responsável]

### 6. Continuidade
- Último checkpoint:
- Próximo arquivo/playbook:
- Lacuna que pode mudar a decisão:
```

## Regras

- Não esconder dado frágil.
- Não transformar status em relatório longo.
- Se houver red line, abrir por ela.
- Se for retomada de frente longa, cruzar com `playbooks/session-continuity.md`.
- Status não substitui relatório executivo nem reconciliação de dados.
