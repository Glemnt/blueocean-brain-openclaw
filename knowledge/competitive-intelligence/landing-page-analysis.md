# Landing Page Competitive Analysis

Framework canônico para analisar landing pages de concorrentes e referências de mercado com utilidade real para a Blue Ocean.

## Princípio
Landing page competitiva não deve ser lida só pela estética.

A análise precisa distinguir:
- o que é posicionamento
- o que é mecanismo de conversão
- o que é prova
- o que é filtro de público
- o que parece forte, mas é só superfície

## Camadas de análise
### 1. Promessa principal
- qual transformação a LP promete?
- a promessa é específica, nichada ou genérica?
- a promessa está ancorada em dor, oportunidade, previsibilidade ou autoridade?

### 2. Público e filtro
- para quem a página parece falar?
- ela filtra bem o ICP ou tenta servir para todo mundo?
- a mensagem exclui o público errado de forma saudável?

### 3. Oferta e mecanismo
- qual é a oferta principal?
- existe mecanismo claro ou só promessa ampla?
- a LP vende diagnóstico, call, reunião, demonstração, material ou proposta?

### 4. Prova e credibilidade
- que tipo de prova aparece?
- a prova é específica, quantificada, contextualizada ou decorativa?
- há sinais de credibilidade operacional real?

### 5. Estrutura de conversão
- a CTA é clara?
- a sequência da página parece conduzir para conversão ou só informar?
- existe fricção excessiva ou fricção estratégica de qualificação?

### 6. Copy e narrativa
- o texto trabalha dor, mecanismo, objeção, oportunidade ou comparação?
- a narrativa é coerente do início ao fim?
- a LP parece escrita para convencer ou só para parecer profissional?

### 7. Leitura estratégica
- o que a Blue Ocean pode adaptar daqui?
- o que não deve copiar?
- isso reforça padrão recorrente ou é caso isolado?

## Perguntas de saída
Uma boa análise de LP deve responder:
1. qual é a promessa central
2. para quem a página parece feita
3. qual é a oferta e o mecanismo
4. que prova sustenta a narrativa
5. se a estrutura converte ou só aparenta qualidade
6. o que vale adaptar
7. o que deve ser evitado

## Alertas comuns
- LP bonita não é sinônimo de LP forte
- prova estética não é prova estratégica
- páginas genéricas podem parecer sofisticadas e ainda assim filtrar mal
- não inferir performance real só pela qualidade aparente da página

## Regra final
A melhor leitura de LP competitiva não pergunta só se a página é bonita.

Pergunta se ela parece construída para persuadir o público certo com mecanismo, prova e direção.
