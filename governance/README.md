## Papel desta camada
Toda regra global deve morar aqui antes de ser referenciada por playbooks ou subagents.

A governança não serve apenas para travar erro. Ela também serve para:
- definir alçada
- padronizar confiança
- impedir decisões irreversíveis com base frágil
- preservar ownership claro
- garantir que aprendizados importantes virem inteligência consolidada
